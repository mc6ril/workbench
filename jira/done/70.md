## Description

Create a reusable DroppableZone component in `presentation/components/ui/DroppableZone.tsx` for creating drop targets in drag-and-drop scenarios. The component must follow project conventions and include full accessibility support with keyboard navigation and ARIA attributes.

## User Story

As a developer, I want a reusable DroppableZone component so that I can easily create drop targets for drag-and-drop functionality with proper accessibility and keyboard navigation support.

## Acceptance Criteria

1. **Component Structure**
   - [ ] DroppableZone component created in `presentation/components/ui/DroppableZone.tsx`
   - [ ] Component follows project conventions (arrow function, export default, `type` for props)
   - [ ] Component accepts props: `children`, `id`, `onDrop`, `accepts` (optional), `disabled`, `ariaLabel`
   - [ ] Component uses SCSS variables from `styles/variables/*` for all styling values
