## Description

Create a reusable DraggableItem component wrapper in `presentation/components/ui/DraggableItem.tsx` for enabling drag-and-drop functionality on components. The component must follow project conventions and include full accessibility support with keyboard navigation and ARIA attributes.

## User Story

As a developer, I want a reusable DraggableItem component wrapper so that I can easily make any component draggable with proper accessibility and keyboard navigation support.

## Acceptance Criteria

1. **Component Structure**
   - [ ] DraggableItem component created in `presentation/components/ui/DraggableItem.tsx`
   - [ ] Component follows project conventions (arrow function, export default, `type` for props)
