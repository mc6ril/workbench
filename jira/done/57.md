## Description

Create a reusable EmptyState component in `presentation/components/ui/EmptyState.tsx` to display empty states for lists, tables, and content areas. The component must follow project conventions and include full accessibility support with proper ARIA attributes and helpful messaging.

## User Story

As a user, I want to see helpful empty state messages when there is no content to display so that I understand the current state and know what actions I can take.
