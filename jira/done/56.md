## Description

Create a reusable ErrorMessage component in `presentation/components/ui/ErrorMessage.tsx` to display error messages throughout the application. The component must follow project conventions and include full accessibility support with proper ARIA attributes and screen reader announcements.

## User Story

As a user, I want to see clear error messages when something goes wrong so that I understand what happened and how to fix it.
