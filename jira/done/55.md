## Description

Create a reusable Loader component in `presentation/components/ui/Loader.tsx` to display loading states throughout the application. The component must follow project conventions and include full accessibility support with proper ARIA attributes and screen reader announcements.

## User Story

As a user, I want to see a loading indicator when content is being fetched so that I know the application is working and my request is being processed.
