## Description

Create a reusable Tooltip component in `presentation/components/ui/Tooltip.tsx` for displaying contextual help and additional information on hover or focus. The component must follow project conventions and include full accessibility support with proper ARIA attributes and keyboard navigation.

## User Story

As a user, I want to see helpful tooltips when hovering over or focusing on elements so that I can get additional context and information without cluttering the interface.

## Acceptance Criteria

1. **Component Structure**
   - [ ] Tooltip component created in `presentation/components/ui/Tooltip.tsx`
   - [ ] Component follows project conventions (arrow function, export default, `type` for props)
   - [ ] Component accepts props: `children` (trigger), `content`, `placement`, `delay`, `ariaLabel`
   - [ ] Component uses SCSS variables from `styles/variables/*` for all styling values
   - [ ] Component SCSS file created in `src/styles/components/ui/Tooltip.module.scss`

2. **Tooltip Behavior**

   - [ ] Shows on focus (for keyboard users)
   - [ ] Hides when mouse leaves or focus is lost
   - [ ] Configurable delay before showing (optional)
   - [ ] Position calculation: tooltip appears near trigger element

3. **Placement Options**
