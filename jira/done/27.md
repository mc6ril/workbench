## Technical Considerations

- **Shared Layer** (`styles/components/`):
  - Create `button.scss` for button base styles
  - Create `input.scss` for input base styles
  - Create `card.scss` for card base styles
  - Use BEM methodology for class naming
  - Import variables from `styles/variables/` using `@use`
  - Use variables for all values (colors, spacing, typography, shadows, etc.)

- **BEM Naming Convention**:
  - Block: `.button`, `.input`, `.card`
  - Element: `.button__icon`, `.card__header`, `.input__label`
  - Modifier: `.button--primary`, `.button--disabled`, `.card--elevated`

- **Variables Usage**:
  - All colors from `styles/variables/colors.scss`
  - All spacing from `styles/variables/spacing.scss`
  - All typography from `styles/variables/typography.scss`
  - All shadows from `styles/variables/shadows.scss`
  - All border radius from `styles/variables/border-radius.scss`
  - All transitions from `styles/variables/transitions.scss`
