## Description

Create a reusable Badge component in `presentation/components/ui/Badge.tsx` for displaying status, priority, and other categorical labels. The component must follow project conventions and include full accessibility support.

## User Story

As a developer, I want a reusable Badge component so that I can display status and priority labels consistently across the application.
