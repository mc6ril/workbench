## Description

Create a reusable Textarea component in `presentation/components/ui/Textarea.tsx` with label and error state support. The component should follow the same patterns as the Input component for consistency. It must follow project conventions and include full accessibility support.

## User Story

As a developer, I want a reusable Textarea component with label and error state so that I can build consistent forms with multi-line text input across the application.

   - [ ] Component follows project conventions (arrow function, export default, `type` for props)

   - [ ] Component uses SCSS variables from `styles/variables/*` for all styling values
